图像标签

img标签属性

属性 属性值 描述
src URL 图像的路径
alt 文本 图像不能显时的替换文本
tittle 文本 鼠标悬停时显示的内容
width 像素 XHTML不支持百分比 设置宽度
height 像素 XHTML不支持百分比 设置高度
border 数字 图像边框的宽度

注意：

1.标签可以有多个属性，必须写在开始标签中，位于标签名的后方

2.属性之间不分先后顺序，标签名与属性，属性与属性之间可以用空格隔开

  1. 采取键值对的形式 key=“value”的格式

绝对路径

要引用的文件在下一级文件夹下，文件名前加子文件夹名称
假设info.html路径是：c:\Inetpub\wwwroot\sites\blabla\info.html
假设index.html路径是：c:\Inetpub\wwwroot\sites\blabla\html\tutorials\index.html
在info.html加入index.html超链接的href应该这样写：html/tutorials/index.html

要引用的文件在上一级文件夹下，文件名前加../

假设index.html路径是：c:\Inetpub\wwwroot\sites\index.html
在info.html加入index.html超链接的代码应该这样写：
index.html
